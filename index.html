Help Quick Steps

Follow each action and check the result before moving to the next one.

Daily Dashboard Check

  1. Open Dashboard.
  2. Check Customers, Frames & Lenses, Orders, and Pending Customers cards.
  3. Use Representative, Place, Status, Year, and Month filters only when you need a smaller view.
  4. Check Upcoming Meetings & Notes for today and next appointments.
  5. Check Revenue, Status Summary, and Revenue by Month before closing the day.

Add Customer

  1. Open Customers.
  2. Use Quick Add for name, phone, place, town, representative, and appointment date.
  3. Use Add Customer when you need prescription, frame, lens, payment, and status details.
  4. Press Submit.
  5. Check the customer appears in the table with the correct status and balance.

Update Customer

  1. Open Customers.
  2. Use filters to find the customer.
  3. Tap Update on the customer row.
  4. Edit prescription, frame, lens, payment, or order status.
  5. Submit and check the table updated correctly.

Pending Customers

  1. Tap Pending Customers on Dashboard.
  2. Review every customer that is not completed or delivered.
  3. Call or check the job status one by one.
  4. Update the customer status after each check.
  5. Return to Dashboard and confirm the pending count changed.

Frames & Lenses

  1. Open Frames & Lenses.
  2. Tap Add Item for a new frame or lens.
  3. Enter item ID, name, price, category, and quantity.
  4. Submit the item.
  5. Check the item appears in the table before using it in Orders.

Create Order

  1. Open Orders.
  2. Select the customer.
  3. Tap frame and lens items to add them to the cart.
  4. Enter cash and discount if needed.
  5. Tap Save & Print Receipt, then check total, cash, balance, and customer name.

Filling App

  1. Open Filling App.
  2. Enter camp defaults first if the same town, place, representative, or date repeats.
  3. Type customer details and press Enter to move field by field.
  4. For status, frame, and lens, type short codes or text and press Enter.
  5. Save the record and check it appears in Pending Records.

Before Sign Out

  1. Check Dashboard pending count.
  2. Check today appointments and notes.
  3. Check unsaved offline filling records if you used offline mode.
  4. Download backup when working with weak internet.
  5. Use Sign Out when all checks are complete.
